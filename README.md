# ardunio_mpu6500
高三專題
LCD1602 顯示器
透過 Arduino接收數據並且顯示目前計算出的 
速度或其他即時資訊
系統架構

MPU6050(速度)→ Arduino → 繼電器 → 線性致動器 →輔助輪（升降）

MPU6050(傾斜)→ Arduino → 判斷傾斜 → 蜂鳴器 → 繼電器 → 線性致動器 → 輔助輪（升降）

LCD1602 → Arduino → 顯示速度

OLED → Arduino → 顯示（？）

紅外線控制器（遙控器） → arduino接收到訊號 → 繼電器 → 線性致動器向下


功能 A：測量速度（速度 → 積分 → 速度）
 → 透過 I2C 傳輸數據到 Arduino
 → Arduino 控制線性致動器的運作（如啟動/停止或速度調整）

功能 B：測量傾斜角（加速度 + 陀螺儀 → 傾斜角度）
 → 透過 I2C 傳輸數據到 Arduino
 → Arduino 判斷傾斜是否超過安全門檻
 → 超過時觸發 蜂鳴器 警示→ 輔助輪馬達也放下

功能 C：監測障礙物(使用超音波感測)
 → 透過 I2C 傳輸數據到 Arduino
  →超音波感測器負責偵測障礙物→數據傳輸障礙物數據透過 I2C 傳輸到 Aduino速度
