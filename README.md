# 自行車智慧輔助輪系統

## 功能概述
本系統以 **Arduino** 為核心，整合多種感測器與致動元件，實現：
- 輔助輪自動升降
- 傾斜安全警示
- 速度判斷
- 距離警示
- 系統狀態顯示

系統採用 **OLED 單螢幕顯示設計**，取代 LCD1602，使 UI 更簡潔，可呈現動畫、進度條及即時資訊。

---

## 系統主要功能

### 1. 輔助輪自動升降控制
- 使用 **繼電器 + 線性致動器** 控制輔助輪上下動作。
- 可透過 **紅外線遙控器** 手動控制升降。
- 當偵測到速度或傾斜異常時，自動下降輔助輪。
- OLED 顯示升降狀態（上升 / 下降）。

### 2. 速度偵測（霍爾感測器）
- 霍爾感測器讀取磁鐵訊號，Arduino 計算轉速與速度變化。
- 判斷是否需放下輔助輪（如急煞）。
- OLED 顯示速度狀態（系統邏輯提示，不顯示數字）。

### 3. 傾斜角度偵測（MPU6050）
- 利用陀螺儀與加速度計計算傾斜角度。
- 超過預設安全角度時：
    - 蜂鳴器警示
    - 輔助輪自動下降
    - OLED 左上角即時顯示角度

### 4. 紅外線遙控控制
- 使用者可按 IR 按鍵手動控制輔助輪升降。
- OLED 顯示觸發來源（IR Control）。

### 5. OLED UI 顯示系統
- 系統啟動畫面與自我檢測 Progress Bar
- 即時傾斜角度
- 輔助輪升降圖示
- IR 觸發提示
- 緊急煞車狀態顯示
- 其他感測器觸發事件來源

### 6. 超音波距離警示
- 測量前方距離過近時觸發蜂鳴器。
- 適用於停車或慢速段的安全輔助。

### 7. PWM 線性致動器速度調整
- 使用 PWM 改變線性致動器速度，使輔助輪升降平順。
- 微調升降速度，避免機構震動。
